Este repositorio contiene el código fuente de mi portafolio personal interactivo y currículum vitae web. Diseñado desde cero bajo una filosofía minimalista orientada al rendimiento, sin depender de pesados *frameworks* de frontend, e implementando estética "hacker/gamer" con detalles modernos.

## 🚀 Características Principales

*   **Estética Terminal/Gamer:** Paleta de colores cálida-oscura y neón verde. Efecto de "foco de luz plana" (*spotlight*) reactivo al ratón, inclinación de tarjetas 3D y secuencias de barras de habilidades (XP).
*   **Traducción Dinámica *(100% Vanilla JS)*:** Sistema de internacionalización integrado *(ES/EN)* que altera los descriptores de la web en tiempo real sin tiempo de carga y persistencia vía `localStorage`.
*   **Visibilidad Reactiva:** Interacción inteligente en la pestaña. El favicon animado (con colores neutros/ámbares) y el título varían en tiempo real respondiendo bajo un "¿Ya te vas?" cuando se minimiza o cambia de pestaña.
*   **Currículum Físico/PDF:** Un diseño espejo purificado (`cv.html`) sincronizado usando los mismos tokens de diseño CSS, formateado especialmente para papel A4 y optimizado para conversores de navegador.

## 🛠 Entorno de Tecnologías

*   **HTML5 Semántico**: Con marcas claras para una estructura accesible de cara a los navegadores.
*   **CSS3 (Variables HSL/Animations)**: Motor de colores con variables nativas `--bg`, `--neon`, `--accent` e interactividad hover en el layout.
*   **JavaScript ES6+**: Múltiples lógicas para menús *offcanvas*, validadores para `IntersectionObserver` y sistema de i18n, sin dependencias.
